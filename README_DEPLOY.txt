Dose & Drip Calculator

Final deployable files:
- index.html
- dose-drip-logo.svg
- Dose-Drip-Calculator-final.zip

How to upload:
1. Open your hosting platform dashboard.
2. Upload the ZIP file directly if the platform accepts ZIP uploads.
3. If the platform expects files instead of a ZIP, extract the ZIP and upload index.html and dose-drip-logo.svg.

Recommended static hosts:
- Cloudflare Pages
- Netlify Drop
- GitHub Pages

Notes:
- This project is now prepared as a static site.
- No Node.js server is required for deployment.
- The app includes in-page review status indicators for reviewed, partially reviewed, and pending drug records.
